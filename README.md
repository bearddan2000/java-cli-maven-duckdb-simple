# java-cli-maven-duckdb-simple

## Description
Reads from csv and json giving field names.

## Tech stack
- java
- maven
  - duckdb

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
